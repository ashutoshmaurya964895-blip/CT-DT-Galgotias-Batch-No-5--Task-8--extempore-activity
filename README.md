## Task#08 – Extempore Activity (Design Thinking Practicing for Co-Curricular Skills)

## 📋 Project Objective
This repository documents our structured speech analysis matrices and outline assets compiled for Task 8 under the Design Thinking Practicing framework. This module explores conversational communication structures, real-time articulation layouts, and system domain deep-dives covering information technology's scaling footprint across core industrial logistics systems.

## 🗣️ Analyzed Core Presentation Topics
Our speaking analysis frameworks are segmented into three distinct technology sectors:
* **IT in Automobile:** Exploring connected vehicle frameworks, autonomous processing nodes, telemetry data streams, and onboard diagnostic integration layouts.
* **IT in Metro Rail:** Mapping automatic train operation loops, automated signaling systems, station telemetry analytics, and centralized traffic scheduling frameworks.
* **IT in Avionics:** Tracing mission-critical cockpit flight telemetry, aircraft communication addressing systems, navigational fly-by-wire arrays, and real-time radar routing links.
